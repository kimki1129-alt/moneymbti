# MoneyMBTI

맞벌이 부부의 카드 실적, 연말정산, 생활비, 자산관리, 주식 MBTI를 다루는 정적 콘텐츠 사이트입니다. 애드센스 승인과 검색 유입을 고려해 기본 신뢰 페이지, 카테고리, 콘텐츠 글, robots.txt, sitemap.xml 구조를 포함합니다.

## 로컬에서 보기

`index.html` 파일을 브라우저에서 열거나, 간단한 로컬 서버로 확인합니다.

## Cloudflare Pages 배포

1. 이 폴더를 GitHub 저장소에 올립니다.
2. Cloudflare Pages에서 새 프로젝트를 만들고 저장소를 연결합니다.
3. 빌드 명령은 비워두고 출력 폴더는 `/`로 둡니다.
4. 배포 후 `https://moneymbti.com` 도메인을 Cloudflare Pages에 연결합니다.

## 애드센스 전 체크

- 소개, 문의, 개인정보처리방침, 투자 고지 페이지 연결
- 본문이 얇지 않도록 글 10개 이상 확장 권장
- 광고 삽입 전에도 방문자가 읽을 만한 원문 콘텐츠 확보
- 특정 종목 매수·매도 권유 표현 피하기
